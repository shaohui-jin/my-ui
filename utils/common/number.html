getRandom ​

获取指定范围内的随机整数

配置参数 ​

ts
declare const getRandom: (min: number, max: number) => number;
1
参数名类型默认值描述
minnumber最小值
maxnumber最大值

示例 ​

js
getRandom(1, 10) // 返回1到10之间的随机整数
